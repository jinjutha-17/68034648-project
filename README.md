# ระบบจัดการอพาร์ทเม้นท์ (Apartment Management System)
**โปรเจกต์โดยมหาวิทยาลัยศรีปทุม (SPU)**

## 📝 บทนำ (Introduction)
ในปัจจุบันการบริหารจัดการหอพักมีความซับซ้อน ระบบนี้จึงถูกพัฒนาขึ้นเพื่อแก้ปัญหาการจัดเก็บข้อมูลที่ล่าช้าและความผิดพลาดจากการบันทึกด้วยมือ โดยช่วยให้การจัดการข้อมูลห้องพัก ผู้เช่า การคำนวณค่าสาธารณูปโภค และการแจ้งซ่อม เป็นไปอย่างมีระเบียบและตรวจสอบย้อนหลังได้ง่าย

## ✨ คุณสมบัติเด่น (Features)
ระบบครอบคลุมการทำงานหลักดังนี้:
* **จัดการข้อมูลผู้เช่าและห้องพัก:** ตรวจสอบสถานะห้องว่าง/ไม่ว่าง และบันทึกประวัติผู้เช่า
* **ระบบคำนวณค่าสาธารณูปโภค:** บันทึกมิเตอร์น้ำ-ไฟ และคำนวณค่าใช้จ่ายอัตโนมัติ
* **การออกใบแจ้งหนี้และการชำระเงิน:** สร้างใบแจ้งหนี้ (Invoice) และบันทึกการชำระเงิน (Payment)
* **ระบบแจ้งซ่อม (Repair Tracking):** ติดตามสถานะการซ่อมแซมภายในห้องพัก
* **การจัดการพัสดุ (Parcel Management):** บันทึกการรับพัสดุของลูกบ้าน

## 🗄️ โครงสร้างฐานข้อมูล (Database Schema)
โปรเจกต์นี้ใช้ระบบฐานข้อมูล **SQLite** โดยประกอบด้วยตารางหลักๆ ดังนี้:
* `TENANT`: ข้อมูลผู้เช่า
* `ROOM`: รายละเอียดห้องพักและสถานะ
* `REPAIR`: ข้อมูลการแจ้งซ่อม (เชื่อมโยงกับ `room_id`)
* `PAYMENT`: บันทึกการชำระเงิน
* `PARCEL`: ระบบจัดการพัสดุ
* `METER`: จัดเก็บค่ามิเตอร์น้ำ-ไฟ
* `INVOICE`: รายละเอียดใบแจ้งหนี้ประจำเดือน

## 🛠️ เทคโนโลยีที่ใช้ (Tech Stack)
* **Database:** SQLite
* **Language/Framework:**  Python, PHP, Java,HTML/CSS
* **Tools:** SQLite, VS Code
